## 光标寄存器



光标位置信息位于显卡的2个光标寄存器中，总共16位，分为高位和低位存储。

例如。

标准VGA模式为 25 行 80 列

pos = 0 ：表示 位于 1 行 0 列

pos = 27 ：表示 位于 2 行 2 列

pos =1999 ： 最右下角





读取光标值



| 端口 | 值  | 读写 |
| ---- | ---- |---- |
| 0x3d4 | 0xe0 | 写入 |
| 0x3d5 | 光标高8位值 | 读取 |
| 0x3d4 | 0xf0 | 写入 |
| 0x3d5 | 光标低8位值 | 读取 |
